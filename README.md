# Brute Force Detection Lab (Mini SOC)

## Objective
Simulate and detect brute force attacks in a mini SOC environment using Wazuh SIEM.

## Lab Architecture
- Attacker: Kali Linux
- Victim: Windows Machine
- SIEM: Wazuh

## Tools Used
- Wazuh (SIEM)
- Kali Linux
- Windows
- Sysmon (for logging)
